RUNNING THE EXPERIMENT
======================

--- Machine A (host) ---

  Terminal 1:
    cd Social_Gaze_Experiment
    bash start_lab.sh

  Terminal 2:
    cd Social_Gaze_Experiment
    .venv/bin/python data/SendData.py

start_lab.sh will print the URL — open it in Machine A's browser.

--- Machine B ---

  Terminal 1:
    cd Social_Gaze_Experiment
    git pull
    .venv/bin/python data/SendData.py

  Then open Machine A's URL in Machine B's browser.

--- Do not use the GitHub Pages URL in the lab ---
